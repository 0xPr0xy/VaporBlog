<h1>Vapor Blog Framework written in Swift</h1>

<h2>Features</h2>
<ul>
    <li> Authentication </li>
    <li> Search </li>
    <li> Admin Page creation </li>
    <li> Admin Article creation </li>
    <li> Google Analytics on Public Pages </li>
    <li> Disqus on Public Articles </li>
</ul>
